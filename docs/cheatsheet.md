# ターミナル操作チートシート

ユースケースから層をまたいでキーを引くための一覧です。表は「キー・操作名・一行の用途」に限定し、挙動の詳細は書きません（鮮度規約は [AGENTS.md](../AGENTS.md) を参照）。

**迷ったら `Alt-X`。** コマンドパレット（[fzf_command_palette](../fish/functions/fzf_command_palette.fish)）は定義がそのまま一覧になる設計で、fish レベルの操作については常にこの表より新しい正です。

## 前提と定義元

| 層      | 定義元                                                                                 | 備考                                                 |
| ------- | -------------------------------------------------------------------------------------- | ---------------------------------------------------- |
| Ghostty | [ghostty.nix](../nix/modules/home/programs/ghostty.nix)                                | ほぼデフォルト。テーマと macos-option-as-alt のみ    |
| herdr   | [herdr.nix](../nix/modules/home/programs/herdr.nix)                                    | prefix = `Ctrl+B` の順番押し。ペイン内アプリより優先 |
| fish    | [fzf-widgets.fish](../fish/conf.d/fzf-widgets.fish) + [functions/](../fish/functions/) | fzf 本体のデフォルトは Home Manager の fzf 統合由来  |

## 探す・移動する

| キー     | 操作                 | 用途                                |
| -------- | -------------------- | ----------------------------------- |
| `Ctrl+]` | fzf_ghq_repos        | ghq リポジトリ + gtr worktree へ cd |
| `Alt+Z`  | fzf_zoxide_jump      | よく行くディレクトリへジャンプ      |
| `Ctrl+R` | fzf 履歴検索         | コマンド履歴から選んで再実行        |
| `Ctrl+T` | fzf ファイル検索     | カレント配下のファイルパスを挿入    |
| `Alt+C`  | fzf ディレクトリ移動 | カレント配下のディレクトリへ cd     |
| `gtr cd` | （コマンド）         | worktree 一覧から移動               |

## Git を操作する

| キー           | 操作                 | 用途                                                             |
| -------------- | -------------------- | ---------------------------------------------------------------- |
| `Ctrl+O`       | fzf_git_branches     | ブランチを切り替える                                             |
| `prefix+alt+g` | lazygit ポップアップ | diff・ログ・staging をサッと確認                                 |
| `prefix+alt+r` | reviewr トグル       | エージェントの成果物を隣ペインでレビューし、行コメントを送り返す |

## worktree を扱う

詳細な使い分けと運用ルールは [worktree ワークフロー](worktree.md)を参照。

| キー             | 操作                    | 用途                                                            |
| ---------------- | ----------------------- | --------------------------------------------------------------- |
| `gwn`            | （パレット / コマンド） | ブランチを選んで worktree を作成（herdr 内はワークスペース化）  |
| `gwo`            | （パレット / コマンド） | 既存 worktree を herdr ワークスペースとして開く（herdr 内限定） |
| `gwd`            | （パレット / コマンド） | worktree を選んで削除（複数可）。先にワークスペースを閉じる     |
| `prefix+shift+g` | open_worktree           | 既存 checkout をワークスペースとして開く                        |

## herdr を操作する

カスタムキーの背景は [herdr 運用ガイド](herdr.md)を参照。デフォルトキーの全量は `prefix+?`。

| キー                          | 操作                            | 用途                                     |
| ----------------------------- | ------------------------------- | ---------------------------------------- |
| `prefix+?`                    | ヘルプ                          | 現在のバインド一覧を表示                 |
| `prefix+alt+s`                | sessionizer                     | ghq プロジェクトを選んでワークスペース化 |
| `prefix+w`                    | ワークスペース選択              | ワークスペースを切り替える               |
| `prefix+c` / `prefix+1..9`    | タブ作成 / 切替                 | タブ操作                                 |
| `prefix+v` / `prefix+-`       | ペイン分割                      | 縦 / 横に分割                            |
| `prefix+h/j/k/l`              | ペイン移動                      | フォーカスを移す                         |
| `prefix+z`                    | ズーム                          | ペインを全画面トグル                     |
| `prefix+x` / `prefix+shift+d` | ペイン / ワークスペースを閉じる | 後者は worktree 削除の前に               |
| `prefix+b`                    | サイドバー                      | エージェント一覧の表示をトグル           |
| `prefix+o`                    | 通知先へ移動                    | 直近の通知を出したペインへ飛ぶ           |
| `prefix+q`                    | デタッチ                        | セッションを残して離脱                   |
